Course Levels

Level 1 — C# Fundamentals

Learn the foundations of C# programming: variables, data types, conditions, loops, collections, functions, and how to think in code. Target: complete beginners.

  • Introduction to .NET & Why C#
  • Variables, Data Types & Memory
  • Conditions, Loops & Flow Control
  • Functions, Methods & Parameters
  • Collections (List / Dictionary / Array)
  • Error Handling & Debugging Basics
  • Basic Problem Solving Challenges

Level 2 — OOP & Advanced C#

Object-Oriented Programming: classes, objects, inheritance, polymorphism, interfaces, SOLID basics, and clean code principles to write maintainable software.

  • OOP Mindset — Thinking in Objects
  • Classes, Objects & Encapsulation
  • Inheritance, Polymorphism & Interfaces
  • Abstract Classes vs Interfaces
  • SOLID Principles (Intro)
  • Generics Basics
  • Clean Code & Naming Standards

Level 3 — SQL Server + LINQ + EF Core Basics

Database design, SQL queries, joins, stored procedures, LINQ, and Entity Framework Core (code-first, migrations, CRUD operations).

  • Database Design + ERD
  • SQL SELECT / INSERT / UPDATE / DELETE
  • Joins, Subqueries & Aggregations
  • Stored Procedures Fundamentals
  • LINQ Filtering & Projection
  • EF Core Code-First + Migrations
  • CRUD with Repository Pattern (intro)

Level 4 — ASP.NET Core MVC

MVC architecture, controllers, Razor views, validation, identity roles, file uploads, pagination, filtering, and the main differences between .NET Framework vs .NET Core.

  • .NET Framework vs .NET Core Differences
  • MVC Architecture
  • Controllers, Views & Models
  • Basics of HTML, Css, Js, jQuery and Ajax
  • Razor Views + Validation
  • Identity Authentication & Roles
  • File Uploads
  • Pagination / Filtering / Sorting

Level 5 — ASP.NET Core Web API + JWT Authentication

REST API development, DTOs, AutoMapper, middleware pipeline, JWT authentication & refresh tokens, API versioning, Swagger documentation, and building a real production-grade backend API.

  • REST API Fundamentals
  • DTOs + AutoMapper
  • Routing + API Versioning
  • Middleware Pipeline
  • JWT Authentication (Register / Login)
  • Role-based Authorization
  • API Documentation with Swagger

Frequently Asked Questions

Do I need any previous programming experience?
No. Level 1 starts from zero. You only need a laptop and commitment. Everything else will be explained step by step.
Are the sessions live or recorded?
Sessions are live on Zoom or Google Meet. Each week we have 2 sessions: one for explaining concepts and one for implementation + live Q&A. All sessions are recorded and provided for replay.
How many students in each group?
Maximum of 10 students per group. This ensures you get personal attention, feedback, and support.
Do I need a powerful laptop?
Not really. Any normal Windows laptop is fine. We will install Visual Studio Community and SQL Server.
Will I build real projects?
Yes. Each level has practical implementation, and Level 5 ends with a full production-grade backend project using ASP.NET Core Web API.
